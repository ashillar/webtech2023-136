Photo Posting API Documentation

Endpoint

POST /photos

Request Body

The API request must include the following parameters:

  • author (string, required)
  • alt (string, required)
  • tags (string, optional)
  • image (string, required) - URL of the photo to be posted
  • description (string, optional) - description of the photo

Response

On success, the API will return a 201 status code.

On error, the API will return a console error with the error message.

Example

Below is an example of a successful photo post: 

        POST /photos
        {
        "author": "John Doe",
        "alt": "A beautiful sunset",
        "tags": "sunset",
        "image": "https://example.com/sunset.jpg",
        "description": "A beautiful sunset over the ocean."
        }
        Response:
        201
