  • Guideline Improvement
    • Guideline 1.1: We added alt attribute to the images which can be seen in script.js file.
    • Guideline 1.4: We changed the color of the links so it is more visible.
    • Guideline 2.4: There are titles that describe parts of the web page. The 'about', 'modal' and 'report' titles all describe what they are and send you to the intended section for the web page. You can also scroll down and at the sections for the news and the other sections, there is also a heading above it saying what it is.
  • Functionality
    • User are able to search the data of the authors by typing their names from the search box
    • User are able to add data to the table from clicking modal on the navigation bar or clicking submit button at the bottom of the table. However, going back to the html page and reloading the page is needed to update the table.
    • User are able to reset the data on the table and the server by clickling reset database button. However, reloading the page is needed to update the table
    • User are able to filter data on specific author by clickling the button on the author's name. After the table is filtered, click again to show the original table
